GREENS

Live

Greens is a landing-page built using Bootstrap. The landing page has a clean and modern design, with a hero section at the top that features a large banner image. Below the hero section, the landing page incorporates several sections to highlight key information about the organization. These sections are designed using Bootstrap's grid system, ensuring responsive layouts across different screen sizes.

As a frontend developer, my role involved coding the HTML structure, implementing Bootstrap's CSS classes and components to achieve the desired layout and styling, and ensuring a consistent user experience across different devices and screen sizes.
